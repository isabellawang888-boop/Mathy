# Mathy
A simple project, helps you to improve mental calculation skill, created by a 12yrs old girl - python newbie:)

Hi guys, welcome to mathy, a very good learning tool to use for math improvement on mental calculation!

In this program, you will have to choose 2 system, quality and quantity, choosing those 2 different system will affect the question you need to complete per day, the level of mental calculation and the speed required.

The operation you will be needing to perform is addition, subtraction, and multiplication.

The quality and quantity system both have their own daily mission, if you complete the requirement for the daily mission, you will earn a star.

Have fun!
